# DeedForge

> Project is currently in active development.

DeedForge is a Rust learning project that models a blockchain-style real estate deed registry.

The goal of the project is to practice Rust by building a small domain model around property ownership, document hashes, ownership transfers, and future listing workflows for sale or rent.

Current MVP scope:

- Register property deeds.
- Store owner wallet identifiers.
- Store document hashes.
- Validate required deed fields.
- List registered deeds.
- Transfer ownership.
- Reject unauthorized transfers.
- Keep transfer history.

Planned next step:

- Add listing support for property sale and rent workflows.
